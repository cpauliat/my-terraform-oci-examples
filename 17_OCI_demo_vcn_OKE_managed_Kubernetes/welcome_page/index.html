Terraform on OCI demo for OKE (Kubernetes cluster in OCI)

This demo web page is hosted on a Kubernetes pod in an OKE cluster on OCI.

The following OCI objects were provisioned by Terraform:
- 1 Virtual Cloud Network with 1 public regional subnet + 1 Internet gateway + 1 route table + 1 security list
- 1 OKE cluster containing a node pool with 2 Oracle Linux 7.x nodes.

The Kubernetes pod / deployment and service were deployed using HELM.
